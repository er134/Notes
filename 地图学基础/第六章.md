# 地理信息的符号模型

|符号类型|表示方法|
|-|-|
|点状符号|定名符号法|
|线状符号|线状符号法|
|面状符号|质地法、等值线法、定位图表法、分区统计图法、等值区域法，点值法，范围法|

**定性数据：**
|方法|数据特征|使用量表|视觉变量|
|-|-|-|-|
|定名符号法|0维|定名量表法|形状->色相->网纹->方向|
|线状符号法|1维|定名量表法|形状和颜色|
|质底法（全局连续）|2维|定名量表法|色彩|
|范围法（局部连续）|2维|定名量表法|色彩和网纹|

**定量数据：**
|方法|数据特征|使用量表|视觉变量|
|-|-|-|-|
|定位符号法|0维|顺序或间隔/比率量表|尺寸和颜色|
线性符号|||尺寸、亮度和彩度
点值法（离散分布）|2维|顺序或间隔/比率量表|
